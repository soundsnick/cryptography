# Basic cipher methods on C++

This folder contains algorithms listed below:
- *Diffie-Hellman* cipher algorithm
- *ElGamal* cipher algorithm

## Usage

Run  script using your c++ interpreter.
```bash
$ g++ <method>.cpp
```

## Version
This files use version VC++ 12.0
